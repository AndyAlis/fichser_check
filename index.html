О сервисе

Наш сервис позволяет проверить на оригинальность и наличие гарантии лыжи Fischer по коду с наклейки на лыжах или по серийному номеру*. Проверка доступна для лыж: Fischer Helium, Fischer Speedmax 3D 61k, Fischer Speedmax 3D.

По коду с наклейки По серийному номеру*

* На данный момент проверка по серийному номеру не доступна.
